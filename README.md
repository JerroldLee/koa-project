=======================

`执行方式 node -harmony app`

官网依赖模块包括:

>       	"koa": "*",//koa核心模块
        "koa-route": "*",//路由模块
        "koa-static": "*",//静态文件加载
        "co":"*",//异步流
        "co-fs": "*", //文件流
        "co-body": "*",//post JSON模块
        "co-views": "*",//视图模块
        "koa-compose":"*",//函数合并执行
        "swig": "*",//模版引擎
        "xss":"*",	//方式xss 攻击
        "mongoose":"*"//mongo链接库

文件以及作用：
> static.js 解决windows开发中遇到的问题以及解决方案
> yield.js 测试 异步流
> run.md 运行命令
